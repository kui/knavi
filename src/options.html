Basic Settings

Hold the Peek Key to show hints, type a hint's letters to select an element, then release the Peek Key to click it. Hold Ctrl, Shift, or Alt on release to open links in a new tab or window. Hint Letters are the characters used to label hints.

The Peek Key is optional — press [Unbind] to disable it and use Sticky Mode (below) instead.

Blur Key

A helper that deactivates the currently focused element, such as an input, textarea, or iframe. Press [Unbind] to disable it.

Sticky Mode

An alternative to holding the Peek Key: press the Sticky Key to show hints and keep them visible without holding a key down, then press the Action Key to click the selected element or the Cancel Key to dismiss. Modifiers (Ctrl/Shift/Alt) on the Action Key open links in a new tab or window.

The Action and Cancel Keys also work during Peek Key sessions. Press [Unbind] on any field to disable it.

Cycle Key

When the selected hint chip is overlapped by other chips, press the Cycle Key to move the selection through those overlapping hints without typing more letters. Press [Unbind] to disable.

Style

Styles for the hint elements and overlays.

Black List

Disable knavi on the listed pages. Supports wildcard patterns.

Additional Selectors

Extra CSS selectors for hintable elements on specific pages. Keys are wildcard URL patterns; values are selectors or arrays of selectors. Parsed as JSON5.
