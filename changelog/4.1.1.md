# Version 4.1.1

Silence benign console warnings from racing iframe and tab lookups.

- Fix
  - Downgrade `FrameIdAnnouncement from unknown source` from `warn` to `debug`.
    The `source.parent === window` guard couldn't distinguish real lookup
    misses from benign cases (iframes inside closed shadow roots, React
    remount races losing `contentWindow` identity), producing noisy console
    warnings on SPA pages without any functional impact.
  - Suppress `Unchecked runtime.lastError: No tab with id: X` warnings from
    `chrome.action.setIcon` / `chrome.tabs.get` racing with tab removal.
    The Promise form of `setIcon` resolves successfully while still setting
    `lastError` internally, so a `.catch()` cannot mark it as checked. Wrap
    both calls in callback form and consume `lastError` synchronously inside
    the callback.
